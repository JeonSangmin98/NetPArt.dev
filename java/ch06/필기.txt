클래스, 메소드 심층

접근 제어
	- 다른 클래스가 특정 필드나 메소드에 접근 하는 것을 제어하는 것
	- public > protected > default > private   (공유 범위)
	
정보 은닉
	- 구현의 세부 사항을 클래스 안에 감추는 것
	
설정자
	- 필드의 값을 설정하는 메소드
	- setXXX()
접근자
	- 필드의 값을 반환하는 메소드
	- getXXX()
	
생성자
	- 객체가 생성될 때 필드에게 초기값을 
	  제공하고 필요한 초기화 절차를 실행하는 메소드
	- 반환타입 없음

매개 변수에
	- 기초형 변수값을 복사
	- 객체의 참조값이 복사
	- 배열 주소 값이 복사
	
정적 멤버
	- static 이 붙어 있는 멤버
	- 모든 객체를 통틀어서 하나만 생성, 모든 객체가 공유
	- 클래스당 하나만 생성
	
인스턴스 변수
	- 인스턴스마다 별도로 생성되는 변수
	
정적 변수(클래스 변수)
	- 모든 객체 공통 변수
	- 하나의 클래스에 하나만 존재
	
정적 메소드
	- static 수식자를 메소드 앞에 붙이며 클래스 이름을 통하여 호출
	- ex) double value = Math.sqrt(9.0);
	
상수
	- 클래스 변수의 대표적인 예
	- 저장공간이 낭비되지 않게 클래스 변수로 정의하는 것이 바람직
