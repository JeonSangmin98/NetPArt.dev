GUI(Graphic User Interface)
	- 컴포넌트들로 구성(입력창,체크박스 등등)
	종류
		- AWT
			- os에 따라 달라짐
		- SWING
			- os 상관없이 일관된 화면
	컨테이너와 컴포넌트
		- 기본 컴포넌트
		- 컨테이너 컴포넌트
			- 다른 컴포넌트 안에 포함할 수 있는 컴포넌트
	최상위 컴포넌트
		- 절대 다른 컨테이너 안에 포함될 수 없는 컨테이너
		- JFrame, JDialog, JApplet 등
	일반 컨테이너
	
	GUI 작성 절차
		1) 컨테이너 생성
		2) 컴포넌트 추가
		
	프레임의 속성 변경
		- 조상 클래스들의 메소드 사용할 수 있음
	
	조상 클래스
		- Component
		- Container
		- Window
		- Frame
			- 자바 GUI의 기초
		- JFrame
			- Frame을 스윙 출시에 맞춰 변경
	기초 컴포넌트
		- 패널 JPanel
			- 컴포넌트들을 포함, 컨테이너 중 하나
			- 화면을 2개로 나누어서 표시 하고자 할 때
		- 레이블  JLabel
			- 편집이 불가능한 텍스트 표시
		- 버튼  JButton
			- 이벤트 발생시켜 원하는 동작 실행
		- 텍스트 필드 JTextField
			- 입력이 가능한 한 줄의 텍스트 필드
			
배치 관리자 설정
	- 생성자를 이용
	- setLayout() 메소드 사용
		- FlowLayout
			- 자동으로 왼쪽에서 오른쪽으로 가로 배치
			- 디폴트 배치 관리자
			- pack()
				- 요소에 맞게 크지 지정		
		- BorderLayout
			- 5개의 영역으로 구분
			- 각각의 영역에 컴포넌트 배치
			- 넣는 순서 상관없이 지정한 곳 배치
		- GridLayout
			- 격자 모양으로 배치
