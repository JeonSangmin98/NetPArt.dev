이벤트-구동 프로그래밍

이벤트 처리 과정
	(1) 버튼 클릭
	(2) 이벤트 발생
	(3) 코드 실행
	(4) 작업 실행
	
이벤트 리스너(event listener)
	- 발생된 이벤트 객체에 반응하여 이벤트를 처리하는 객체
	
처리 과정
	1) 이벤트 리스너 클래스 작성
	2) 이벤트 리스너를 컴포넌트에 등록
	
이벤트 객체
	- EventObject 클래스 상속 받음
	- 발생된 이벤트에 대한 모든 정보를 리스너로 전달
	
이벤트 처리 방법
	- #1 내부 클래스로 처리
	- #2 MyFrame 안에서 이벤트도 처리
	- #3 무명 클래스 사용
	
액션 이벤트
	- 버튼을 클릭하는 경우
	- 메뉴 항목 선택
	
이벤트 발생원의 식별
	- getSource() : 이벤트를 발생시킨 객체 
	- getId() : 타입 식별
	- getActionCommand() : 컴포넌트 이름
	
Key 이벤트
	- KeyListener 인터페이스 구현
	- 키보드에서 입력
	
마우스 이벤트
	- 이동 : MouseMotionEvent
	- 클릭 : MouseEvent
