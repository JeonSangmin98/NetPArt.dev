Calendar,Date

 - Date는 이전 버전용, Calender는 Date보다 향상된 기능 제공
 Calendar cal = Calendar.getInstance();	// Calendar 클래스를 구현한 클래스의 인스턴스 변환
 Date d = new Date(cal.getTimeInMillis());		// Calendar -> Date 변환
 
 Date d = new Date();	
 Cal.setTime(d); 	// Date -> Calendar 변환
