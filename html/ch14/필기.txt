반응형 웹 디자인
    - 내용을 그대로 유지, 다양한 화면 크기에 맞춰서 표시
    - 화면 크기에 반응 하여 요소들을 자동으로 바꾸어 구현 : 반응형 웹 디자인
    - 장점
        - 모든 기기 접속 가능
        - 가로모드 레이아웃 변경가능
        - 유지 관리 용이

    - 단점
        - 최신 모던 웹브라우저에서만 적용

뷰포트
    - 실제 내용이 표시되는 영역
    - 기기 화면에 맞춰 확대/축소해서 내용 표시
    - head 태그 안 meta 태그 에서 지정
    - 일반적으로 뷰포트의 너비를 스마트폰 화면 너비에 맞추고 초기 화면 배율을 1로 지정

가변 그리드 레이아웃
    - 그리드 시스템
        - 여러개의 칼럼으로 나눔, 묶어 배치
        - 화면 너비값에 따라 960 그리드, 1200 그리드 등으로 나뉨
        - 칼럽 개수에 따라 12,16,24 등으로 나뉨
        - 주로 960 픽셀 12칼럼 그리드 사용
        - 고정 그리드 : 너비 고정
        - 가변 그리드 : % 같은 가변 값으로 지정

가변요소
    - 가변 글꼴
        - em 단위
            - 부모 요소 폰트의 대문자 M 너비를 1em으로 지정 1em=16px
        - rem 단위
            - 부모 요소가 중첩을 경우 글자 크기가 계속 달라짐
            - 처음부터 기본 크기 지정
    - 가변 이미지
        - 이미지 너비도 조절됨
        - css이용
            - %로 지정
