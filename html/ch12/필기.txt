연결 선택자
 - 선택자와 선택자를 연결 
 - 적용대상 한정
 - 하위 선택자
    - 모든 하위 요소에 스타일 적용
    - ex) ul li(하위) { }
 - 인접 형제 선택자
    - 같은 부모를 가진 형제 요소
    - ex) h1 + p { }

[속성] 선택자
 - 지정한 속성을 가진 요소를 찾아 스타일 적용

[속성 = 값] 선택자
 - 주어진 속성과 속성 값이 일치하는 요소를 찾아 스타일 적용

[속성 ^= 값]
 - 특정 값으로 시작하는 속성 찾아 스타일 적용

[속성 $= 값]
 - 특정 값으로 끝나는 속성을 찾아 스타일 적용

사용자 동작에 반응하는 가상 클래스
 - :link, :visited, :active, :hover, :focus

UI요소 상태에 따른 가상 클래스
 - :enabled, :disabled  : 사용할 수 있을때와 없을 때
 - :checked : 체크박스에서 선택 했을 때

가상 요소
 - ::firstline, ::after, ::before, ::first
