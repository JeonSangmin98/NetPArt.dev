제어문

조건문, 선택문, 반복문

if문
    - 논리현 데이터가 아닌 다른 형이 오는 경우
        - 0, null, "", undefined -> false
        - 그 외의 값은 true
else
    - if 외 조건

confirm()함수
    - 사용자의 확인을 받을 때 사용
    - confirm('대화상자에 띄울 메세지') 형식으로 사용
    - 입력 받은 정보는 불린으로 처리
    - 확인 : true , 취소 : false

else if문
    - 여러 조건

중첩 if문
    - if(){
        if(){
            코드;
        }
    }


switch문
switch(변수){
    case 값:코드;
    break;
    case : ;
    break;
    default:  ; // case 외 나머지
}

while문
while(조건){
    코드
    증감식
}

for문
for(초깃값;조건식;증감식){

}

break문
    - 반복문을 강제로 종료 할 때

continue문
    - 다음 조건식으로 감(건너뜀)
