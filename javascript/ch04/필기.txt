객체

객체 = 기능과 속성을 가짐

기본형
    - 객체.메소드();
    - 객체.속성; 또는 객체.속성=값;
    
종류
    - 내장 객체
    - 브라우저 객체
    - 문서 객체

내장 객체
    - 참조변수(인스턴스 이름) = new 생성 함수();

수학 객체
    - abs,max,min,round,floor,ceil,random,PI 등등

배열 객체
    - var 참조변수 = new Array();
    - var 참조변수 = new Array(값1,값2,값3,...값n);
    - var 참조변수 = new [값1,값2,값3,.. 값n];

문자열 객체
    - 문자형 데이터를 객체 취급
    - 자바스크립트에서 가장 많이 사용
